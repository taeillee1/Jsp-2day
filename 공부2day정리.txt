해당문서의 문자 인코딩 방식을 결정하는 속성

<link rel="stylesheet" type="text/css" href="https://stackpath.bootstrapcdn.com/bootstrap/3.1.1/css/bootstrap.min.css">
JSP로 부트스랩을 이용하여 CSS스타일을 적용하려고해보았을 때 href에 css/bootstrap.min.css만 쓰고 끝낸다면
계속해서 css 스타일이 적용되지않는 문제가 발생하였다
그리하여 부트스트랩 공식홈페이지의 CDN 주소를 활용하여 적용하였더니 정상적으로 작동하였다.

컨테이너(container)
레이아웃을 만드는 가장 상위요소에 클래스의 속성을 정하여 전체 가로폭을 정하는 것

bootstrap사이트에서 다운받아 저장되어있는 곳에서 내가 원하는대로 기본적인 틀을 만들고 색상변경등의 조정 가능

data-target : 팝업창이 위에서 아래로 내려오게하는 구조

aria-expanded=""
true일 때는  요소가 확장 상태
false일 때는 요소가 축소 상태

aria-haspopup=""
treu일 때는 menu와 동일한 의미를 지닌다.

collapse : 숨김기능을 담당
화면이 축소시켰을 때 내가 설정한 메뉴들을 숨기기 기능을 사용할 때 쓰는 것
